隱私權政策

這是一份供開發設定使用的範本。正式上線前，請填入服務經營者、聯絡方式、保存期限與實際用途，並依適用法規完成法律審閱。

版本日期：2026 年 7 月 22 日

01

資料蒐集範圍

當你使用 LINE 登入會員中心時，本服務會蒐集 LINE 使用者識別碼、顯示名稱與頭像網址，並保存你自行填寫的電話與生日。舊版本曾經授權取得的 Email 可能仍存於既有記錄，新版本不再顯示或更新該欄位。

系統亦記錄會員編號、會員建立時間、最後登入時間、登入次數、會員使用狀態與異動時間、執行異動的管理員 LINE User ID、管理員資格狀態、基本裝置環境、Token 核發時間，以及去除重複寫入所需的請求識別碼，用於維持會員服務；不保存 ID Token 本體。

02

資料使用目的

上述資料僅用於建立與辨識會員、顯示會員資料、維持登入紀錄、提供會員相關服務，以及處理使用者提出的資料權利要求。

03

身分驗證與安全

前端會把 LINE 核發的短效 ID Token 傳送至 Google Apps Script 後台，由後台向 LINE Platform 驗證。系統不會把 ID Token、Access Token 或 LINE 密碼保存到會員試算表。

04

保存、分享與委外服務

會員資料保存於服務經營者管理的 Google 試算表，並使用 Google Apps Script 處理。除履行服務、法令要求或取得你的同意外，不會任意提供給其他第三方。

正式政策應補上明確保存期限、刪除週期，以及 Google 與 LINE 服務適用的資料處理說明。

05

你的資料權利

你可以在登入後的會員中心查看主要會員資料、修改或清空電話與生日，並使用「刪除會員資料」功能刪除該筆會員紀錄。若需查詢、更正或提出其他要求，請聯絡服務經營者。
